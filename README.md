# Generative_AI_Research_49578508

## Project Overview
This repository contains research materials for the study titled
"Using Generative AI Tools – Boon or Bane".
The repository is organised to support effective research data management,
version control, and collaborative academic research.

## Folder Structure
The repository is structured as follows:

- literature_review  
  Contains journal articles, conference papers, books, and other sources used
  for the literature review.

- quantitative_analysis  
  Includes survey data, survey questions, analysis files (e.g. scripts),
  and quantitative analysis reports.

- qualitative_analysis  
  Contains interview transcripts, interview protocols, consent forms,
  qualitative analysis outputs, and data visualisations.
  Sensitive materials such as consent forms require restricted access.

- drafts_and_reports  
  Stores draft research proposals, conference papers, and final research reports.

- additional_materials  
  Includes information sheets, images, and other supporting research materials.

## Version Control and Collaboration
GitHub is used for version control to track changes to research data and
documents over time. Clear naming conventions and meaningful commit messages
are used to maintain transparency and reproducibility.
Future collaborators should follow the existing folder structure and commit
changes regularly.
